# Venue And Visual Contract

## 定位

本 reference 只提供 venue / LaTeX / figure / table / equation 的检查合同。它偏 CV/ML 双栏论文示例，不代表任何会议或期刊的官方模板。

## Venue intake

- 目标 venue、投稿类型、页数、匿名要求、模板来源和截止日期。
- 正文页数、references、appendix、supplementary 的边界。
- 图、表、公式、算法框和代码链接的数量与必要性。

## Visual contract

- 每张 figure 必须有清楚 takeaway、caption、坐标轴、单位和可读字号。
- 每张 table 必须说明数据来源、指标、统计口径和加粗规则。
- 公式必须定义符号，首次出现时解释变量。
- 图表引用顺序要和正文一致。
- 双栏排版要检查溢出、跨栏图、caption、脚注和 citation。

## 输出边界

- 不能承诺自动生成 camera-ready PDF。
- 需要用户提供官方模板或确认采用通用示例 contract。
- 最终 PDF/DOCX/PPTX 成稿和视觉渲染交给 PDF、Documents 或 Presentations。
